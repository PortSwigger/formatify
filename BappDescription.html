This extension provides a user-friendly interface for converting HTTP requests and responses between various formats, making it easier to analyze or reuse them in different tools or contexts. It simplifies the process of generating code snippets or transformed representations directly from Burp Suite.

Features

  • Convert requests and responses into multiple formats (e.g., raw, cURL, Python requests)
  • Supports both HTTP and HTTPS
  • Automatic extraction of headers and body content
  • Customizable output formats
  • Accessible via a context menu within Burp Suite

Usage

  1. Right-click on a request or response in Burp Suite.
  2. Select Formatify → Convert from the context menu.
  3. Choose the desired output format from the available options.
  4. The converted content will be displayed in a pop-up or output window for easy copying.
