Formatify converts HTTP requests into multiple code formats including JavaScript Fetch, cURL, Python Requests, Go, PowerShell, and specialized security testing formats. This all-in-one extension eliminates the need for multiple request conversion tools by providing comprehensive format support in a single interface.

Features

  • Convert requests to JavaScript Fetch, cURL, Python Requests, Python aiohttp, Node.js Axios, Go http, PowerShell, Java OkHttp formats
  • Generate FFUF commands for fuzzing workflows
  • Create CSRF payload builders with automatic HTML form generation
  • Build CORS exploit proof-of-concept code
  • Context menu integration with Repeater and Intruder
  • Dedicated tab interface for manual request input and conversion
  • Copy to clipboard and save to file functionality
  • Background processing with threading support

Usage

  1. In Repeater or Intruder, right-click on any HTTP request
  2. Select "Send to Formatify" from the context menu
  3. Navigate to the Formatify tab to view the imported request
  4. Choose your desired output format from the dropdown menu
  5. Click "Formatify!" to generate the converted code
  6. Use "Copy to Clipboard" or "Save to File" to export the result
